OpenBrowser

AI-Powered Browser Automation

Control your browser with natural language. OpenBrowser is an open-source framework that enables AI agents to browse the web, fill forms, extract data, and automate complex workflows - all through simple text instructions.

# Install OpenBrowser
pip install openbrowser-ai

# Quick start
from openbrowser import Agent
from langchain_openai import ChatOpenAI

agent = Agent(
    task="Search for OpenBrowser on GitHub",
    llm=ChatOpenAI(model="gpt-4o")
)
await agent.run()
